# Single-cell RNA-seq Analysis of Colorectal Cancer

## Overview

This project demonstrates a single-cell RNA-seq analysis workflow for colorectal cancer tumor samples. The analysis includes preprocessing, quality control, normalization, dimensionality reduction, clustering, cell-type annotation, and exploratory marker-gene analysis.

The goal is to demonstrate practical experience with tumor single-cell RNA-seq data, colorectal cancer tumor microenvironment analysis, and reproducible bioinformatics workflows.

## Workflow

1. Load colorectal cancer single-cell RNA-seq data and metadata
2. Create and process Seurat objects
3. Perform quality control using standard single-cell QC metrics
4. Normalize and scale gene-expression data
5. Run PCA, UMAP visualization, and graph-based clustering
6. Annotate major cell populations
7. Explore tumor- and immune-related marker genes
8. Generate visualizations for biological interpretation
